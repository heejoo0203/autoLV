# 기능 상세 명세 (v1 완료 기준)

## 1. 인증
상태: 구현 완료

### 제공 기능
- 회원가입
- 로그인
- 로그아웃
- 현재 사용자 조회(`me`)

### 핵심 정책
- HttpOnly 쿠키 인증
- 비밀번호 정책 검증(길이, 조합, 바이트 길이)
- 이메일 중복 방지

## 2. 개별조회(지번/도로명)
상태: 구현 완료

### 지번 조회
- 입력: 시/도, 시/군/구, 읍/면/동, 산구분, 본번, 부번
- 처리: PNU 생성 -> VWorld 개별공시지가 조회

### 도로명 조회
- 입력: 시/도, 시/군/구, 자음, 도로명, 건물번호
- 처리: 도로명 -> 좌표 -> 지번 -> PNU -> 공시지가 조회

### 결과 처리
- 연도별 최신 데이터 선택
- 최신연도 내림차순 정렬
- 사람이 읽는 주소 요약 생성

## 3. 도로명 선택기
상태: 구현 완료

### 동작
- 시/군/구 선택 후 해당 지역의 사용 가능한 자음만 표시
- 자음 선택 후 해당 자음으로 시작하는 도로명만 표시
- 데이터 소스: `docs/TN_SPRD_RDNM.txt`

## 4. 조회기록
상태: 1차 구현 완료(클라이언트 저장)

### 동작
- 로그인 사용자 기준으로 기록 분리 저장(`ownerKey`)
- 최근순 목록 표시
- 목록 클릭 시 해당 조회 결과 재열람
- 표시 주소는 PNU가 아닌 사람이 읽는 형식으로 렌더링

### 제약
- 브라우저 `localStorage` 기반으로 기기/브라우저 변경 시 동기화되지 않음
- 서버 영구 저장 기능은 다음 단계

## 5. 파일조회 (v1)
상태: 구현 완료

### 5.1 업로드/양식 안내
- 표준 엑셀 양식 다운로드 제공
- 필수/권장 컬럼, 최대 행 수(`10,000행`) 안내 제공

### 5.2 자동 컬럼 매핑/전처리
- 열 순서가 바뀌어도 헤더 기반 자동 매핑 수행
- 유사 헤더 별칭 사전 적용
- 주소 통합 컬럼 파싱 및 시도/시군구/읍면동 정규화 보정
- 행 단위 오류 분리(정상/실패 사유 기록)

### 5.3 비동기 처리/이력
- 업로드 시 작업 생성 후 비동기 처리
- 진행 상태(대기/처리중/완료/실패)와 진행률 표시
- 작업 이력 페이지당 10개 페이징
- 다중 선택 삭제(처리중 작업 제외)

### 5.4 결과 파일 생성
- 원본 파일의 행/컬럼 구조 유지
- 뒤쪽에 연도별 공시지가 컬럼을 최신순 추가
  - 예: `공시지가_2025`, `공시지가_2024`, `공시지가_2023`
- 행 단위 `조회상태`, `오류사유`, 주소요약/PNU 컬럼 추가
- 완료 작업만 다운로드 허용

## 6. UI/UX
상태: 구현 완료

### 반영 사항
- 비로그인/로그인 상태별 네비게이션 분리
- 중앙 네비게이션(`개별조회`, `파일조회`, `조회기록`)

- 모달 드래그 선택 시 오동작 닫힘 방지
- 반응형/배포형 밝은 톤 디자인 시스템 적용
