# 릴리즈 노트 - v1.0.0 (아카이브)

- 릴리즈 버전: `v1.0.0`
- 릴리즈 날짜: `2026-03-02`
- 기준 브랜치: `main`

> 최신 릴리즈 노트는 `docs/10-release-notes-v2.2.1.md`를 참고한다.

## 1. 릴리즈 요약
v1.0.0은 autoLV의 첫 운영 배포 기준 버전이다.  
핵심은 개별조회/인증/파일조회 기능 완성과, 운영 환경에서 발생한 VWorld 연결 장애를 우회할 수 있는 고정 IP 프록시 경로 안정화였다.

## 2. 포함 기능
- 개별조회
  - 지번/도로명 검색
  - 연도별 공시지가 최신순 조회
- 인증/계정관리
  - 회원가입, 로그인, 로그아웃, 내 정보 조회
  - 프로필 수정(닉네임/이미지), 비밀번호 변경, 회원 탈퇴
- 파일조회(v1)
  - 표준 양식 다운로드/업로드
  - 헤더 자동 매핑 + 전처리
  - 비동기 처리 + 진행률 + 결과 다운로드
  - 이력 페이징(10개), 다중 선택 삭제

## 3. 안정화/인프라 변경
- Railway -> VWorld 직접 호출 장애(502/연결 종료) 대응
  - 직접 호출 실패 시 EC2 프록시 경로 재시도
- AWS EC2(Seoul) 고정 IP 프록시 추가
  - 모듈: `infra/vworld-proxy`
  - 토큰 기반 인증(`VWORLD_PROXY_TOKEN`) 적용
- API 오류 모델 보강
  - `VWORLD_DIRECT_AND_PROXY_FAILED`로 직접/프록시 실패 원인 동시 반환

## 4. 운영 환경 기준
- Web: Vercel
- API: Railway
- DB: Railway PostgreSQL
- Proxy: AWS EC2 + Elastic IP (선택/권장)

## 5. 확인 체크리스트 (릴리즈 직후)
1. `GET /health` 200 확인
2. 회원가입/로그인/로그아웃 확인
3. 개별조회(지번/도로명) 실데이터 응답 확인
4. 파일조회 업로드 -> 완료 -> 다운로드 확인
5. 프록시 우회 경로(`VWORLD_PROXY_URL`) 설정 시 단건조회 정상 여부 확인
