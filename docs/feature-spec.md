# 기능 상세 명세 (v2.2.1)

## 1. 인증/계정
상태: 구현 완료

### 1.1 가입/로그인
- 회원가입(이메일, 비밀번호, 이름, 연락처)
- 이메일 중복 확인
- 이메일 인증코드 발송/검증
- 로그인/로그아웃
- 아이디 저장(브라우저 로컬)

### 1.2 계정 복구
- 아이디 찾기: 이름+연락처 -> 마스킹 이메일
- 비밀번호 찾기: 이메일 인증코드 기반 재설정

### 1.3 마이페이지
- 이름/연락처/프로필 이미지 수정
- 비밀번호 변경
- 서비스 약관 열람(동의 버전/시각)
- 회원 탈퇴(확인 문구 입력)
- 안드로이드 APK 다운로드 버튼

## 2. 개별조회(지번/도로명)
상태: 구현 완료

### 2.1 지번 조회
- 입력: 시/도, 시/군/구, 읍/면/동, 산구분, 본번, 부번
- 처리: PNU 생성 -> VWorld 조회

### 2.2 도로명 조회
- 입력: 시/도, 시/군/구, 자음, 도로명, 건물번호
- 처리: 도로명 -> 좌표/지번 변환 -> PNU -> VWorld 조회

### 2.3 결과 표시
- 연도별 내림차순
- 가격기준년도/토지소재지/지번/개별공시지가/기준일자/공시일자/비고

## 3. 도로명 선택기
상태: 구현 완료

- 지역별 사용 가능한 자음만 노출
- 자음 선택 시 해당 자음 도로명만 노출
- 데이터 소스: `docs/TN_SPRD_RDNM.txt`

## 4. 파일조회
상태: 구현 완료

### 4.1 업로드/가이드
- 표준 템플릿 다운로드
- 필수/권장 컬럼 안내
- 주소 해석 모드(`auto | jibun | road`)

### 4.2 처리 로직
- 헤더 자동 매핑 + 별칭 처리
- 주소 전처리/정규화
- 유니크 주소 키 병렬 조회
- 진행률 업데이트(5행 단위/최소 주기)

### 4.3 결과/이력
- 결과 엑셀 다운로드(완료 작업만)
- 페이지당 10개 이력 표시
- 다중 선택 삭제(처리 중 제외)

## 5. 지도조회
상태: 구현 완료(v2)

### 5.1 입력 방식
- 지도 클릭 조회
- 주소 직접 입력 조회
- 조회기록 복원(`recordId`)

### 5.2 계산/표시
- 현재/전년 공시지가
- 증감률
- 면적
- 면적×단가
- 인근 평균(200m)
- 도로명/지번 주소 표시

### 5.3 추가 조회
- 연도별 공시지가 상세 재조회
- 토지특성 상세(지목/용도지역/용도지구)
- CSV 내보내기

### 5.4 성능
- 클릭 300ms 디바운스
- 동일 좌표 중복 호출 방지
- Redis PNU 캐시
- DB(`parcels`) 가격 캐시

## 6. 구역조회(폴리곤)
상태: 구현 완료(v2.2.1)

### 6.1 분석 흐름
- 지도에서 꼭짓점 선택
- `구역 분석` 실행 -> 미리보기 결과 생성
- 필요 시 필지 선택 제외
- `구역 저장` 클릭 시에만 영속 저장

### 6.2 결과 구성
- 구역명 / 기준연도 / 구역 면적 / 구역 내 필지 수
- 평균 공시지가 / 총 공시지가 합계
- 필지 목록(지번/도로명/면적/공시지가/면적×공시지가)
- 필지 포함 기준 90% 이상 안내 문구

### 6.3 저장 구역
- 우측 사이드 패널에서 목록 확인
- 저장 구역 불러오기 / 이름 수정 / 삭제
- 저장된 구역만 CSV 다운로드 허용

## 7. 조회기록
상태: 구현 완료

- 저장 대상: 개별조회 + 지도조회
- 저장 위치: 서버 DB(`query_logs`)
- 필터: 유형/시도/시군구
- 정렬: 일시/유형/주소/결과건수(3단 토글)
- 선택 삭제: 다중 체크박스 기반 삭제
- 항목 클릭 복원:
  - `jibun|road` -> `/search`
  - `map` -> `/map`
